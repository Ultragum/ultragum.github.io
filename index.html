Projects Overview

PNG Encoder

Developed in Haskell, this PNG encoder takes in raw RGB data and produces valid PNG files. These files are fully compressed, going through each compression algorithm defined within the PNG specification (scanline filtering, LZ77, and Huffman encoding)

Maze Pathfinder

The maze pathfinding project was developed in C++ and utilizes the A* algorithm for traversal. The GUI was created using the wxWidgets library.
